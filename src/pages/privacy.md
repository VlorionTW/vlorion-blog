# 隱私權政策

歡迎造訪由 **Vlorion視界渲染**（以下簡稱「視界渲染」）營運之技術平台 **Vlorion Dev**（以下簡稱「本站」）。視界渲染極為重視用戶的隱私權，並承諾依據相關數據保護法規，保障您的個人資訊安全。本政策旨在說明我們如何處理、儲存及保護您在造訪本站時所產生的相關數據。

### 1. 數據收集之範圍與目的
本站定位為靜態技術文檔、API 規範與開發日誌平台。視界渲染**不會**主動要求您提供敏感個人資料（如姓名、身分證字號、財務資訊或聯繫電話）。

### 2. Cookie 與追蹤技術之應用
為了優化您的瀏覽體驗（例如保存您的深淺色模式偏好、提升頁面載入效率），本站可能會於您的瀏覽器中寫入並讀取 Cookie。
* 您可以隨時透過瀏覽器設定拒絕、限制或刪除 Cookie，此項操作不會影響您閱讀本站的核心內容，但可能需要於每次造訪時重新設定偏好。

### 3. 第三方數據與外部連結
本站文檔可能包含指向外部站點、開源專案庫（如 GitHub）或第三方 API 服務的超連結。這些外部站點均有其獨立的隱私權政策，視界渲染對其內容、營運、數據收集行為或安全性不承擔任何形式的法律責任。

### 4. 數據安全性
視界渲染採用業界標準的技術手段與加密通訊協定（HTTPS），確保本站數據傳輸過程中的安全性，防止文檔內容或傳輸節點遭受惡意篡改。

### 5. 政策之修訂與變更
視界渲染保留隨時依據法規變更或平台運作需求修訂本隱私權政策之權利。修訂後的條款一經於本頁面公布即實施，恕不另行個別通知。
